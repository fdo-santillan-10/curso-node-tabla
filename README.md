# Notas:
Programa en Node para tablas de multiplicar

```
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Base a multiplicar                          [number] [required]
  -l, --listar   Mostrar impresión en consola         [boolean] [default: false]
  -h, --hasta    Límite de números a multiplicar          [number] [default: 10]
```
